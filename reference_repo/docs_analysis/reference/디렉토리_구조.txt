KEY PATTERNS:
=============
1. Camera Labeler Hierarchy: CameraLabeler -> (specific labeler type)
   Each labeler produces AnnotationDefinition + per-frame Annotations

2. Randomization Hierarchy: Scenario -> Randomizers + Parameters + Samplers
   Randomizers query tagged GameObjects and apply variation

3. Sensor Chain: CameraSensor -> CameraChannels (RGB, Depth, etc.) -> post-processing

4. Output Pipeline: Frame data -> DataModel -> MessageBuilder -> IConsumerEndpoint (SoloEndpoint writes JSON)

5. Async Pattern: Labelers return AsyncFuture<Annotation> for expensive computations

IMPORTANT LOCATIONS:
====================
- Labeler interface: /Runtime/GroundTruth/Labelers/CameraLabeler.cs
- Randomizer base: /Runtime/Randomization/Randomizers/Randomizer.cs
- Output interface: /Runtime/GroundTruth/Consumers/IConsumerEndpoint.cs
- SOLO output impl: /Runtime/GroundTruth/Consumers/Solo/SoloEndpoint.cs
- Scenario base: /Runtime/Randomization/Scenarios/ScenarioBase.cs
- Tag manager: /Runtime/Randomization/Randomizers/RandomizerTagManager.cs
